**1. Студенты** 

id            | Фамилия     | Имя  | Отчество     |Группа

 все поля обязательны

**2. Баллы**
 
id    | Предмет |	Баллы за предмет	| id студента 
------|---------|------------------|-------------

 предмет - Enum
 
 id студента - foreign key
